About object states

Table 1.3.1 lists the possible states of components, policies, and Policy Model objects.

Table 1.3.1: Object states

State

Description

Draft

Open for editing. The object has been created and saved, but has not yet been submitted for deployment, or has been taken out of the deployed state for editing.

Deployed

The object has been deployed, and its definition has been sent out to all relevant enforcers.

Deployed Draft

The object has been taken out of the deployed state for editing. An earlier version is deployed.

Inactive

The object has been deactivated, and the deactivated version has been deployed to enforcers.
